# 🍔 Food Delivery Web Application

A full-featured food delivery platform where users can order food, owners manage shops, and delivery partners handle deliveries with real-time tracking.

---

## 🌐 Live Demo

🚀 https://multicartfooddeliverysite-1.onrender.com/signin

---

## 🚀 Features

* 👤 **User Roles**

  * **User** – browse food, place orders, track delivery
  * **Owner** – manage shop, menu, and orders
  * **Delivery Boy** – accept and deliver orders

* 🛒 **Order System**

  * Browse food items
  * Add to cart and place orders
  * Order status updates

* 🗺️ **Live Tracking**

  * Real-time tracking between user and delivery partner
  * Map-based interface
  * Continuous location updates

* 🚚 **Delivery Management**

  * Delivery partners accept/reject orders
  * Track delivery route
  * Update delivery status

---

## 🧠 Project Idea

A system that connects customers, shop owners, and delivery partners in one platform with real-time tracking and role-based functionality.

---

## 🛠️ Tech Stack

* Frontend: HTML / CSS / JavaScript
* Backend: Node.js / Express
* Database: MongoDB
* Maps & Tracking: Google Maps API / Firebase
